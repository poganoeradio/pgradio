Poganoe Radio

Automated podcast content publisher

About

Poganoe Radio is a Russian-language dark humor podcast. This service automatically publishes short video clips from the podcast to social media platforms.

The app uses TikTok Content Posting API to upload and publish podcast video reels directly to the @poganoeradio TikTok account.

How it works

  • Podcast host authorizes via TikTok Login Kit (OAuth 2.0)
  • The app identifies the user via user.info.basic scope
  • Short video clips are generated from podcast episodes
  • Videos are uploaded via Content Posting API (video.upload)
  • Content is published directly to TikTok (video.publish)
